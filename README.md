# C++ Data Structures From Scratch

This repository contains my implementations of fundamental Data Structures in C++, built completely from scratch without using the Standard Template Library (STL).

The goal of this project is to deeply understand memory management, pointers, object-oriented programming, and internal working mechanisms of data structures.
